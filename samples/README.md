# Samples 
## TibboPiOnEdge 
詳細は[こちら](./TibboPiOnIoTEdge)  
速攻で試してみたい方は、[Azure CLI によるデプロイ方法]を参考に、[tibbo-pi-deployment-cli.json](./tibbo-pi-deployment-cli.json) を使って配置すれば動作可能。  
※ファイル内の < Connection String for Azure Storage Account > は、各自の環境に合わせて変更すること。 
配置後、Tibbo-Pi と同じネットワークに接続しているPC等で、  
https://<i>ip-address</i>:1880  
をブラウザで開くと、Node-Redのフローエディタがひょうじされまする。
